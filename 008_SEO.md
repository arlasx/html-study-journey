# HTML

## SEO on-page

- **SEO:**
  - ***Search Engine Optimization* (Otimização para Motores de Busca);**

- **SEO on-page:**
  - **Representa todas as otimizações feitas dentro da página para melhorar posicionamento em buscadores como o Google;**
  - **Envolve:**
    - **Estrutura HTML;**
    - **Conteúdo;**
    - **Performance;**
    - **Acessibilidade;**
    - **Metadados;**




### Estrutura HTML correta (base SEO)

- **Usar HTML semântico:**
  - **`<header>`;**
  - **`<main>`;**
  - **`<section>`;**
  - **`<article>`;**
  - **`<nav>`;**
- **Ajuda buscadores a entender a estrutura do documento;**



### Tag `title`

```html
<title>Curso de HTML5 Completo | Do Básico ao Avançado</title>
```

- **Regras de uso:**

  - **50-60 caracteres;**
  - **Palavra-chave principal;**
  - **Único por página;**
  - **Não repetir o mesmo título em vários páginas;**

- **OBS.:**

  - **O `title` é o fato *on-page* mais forte;**



### Meta `description`

```html
<meta name="description" content="Aprenda HTML5 do zero com foco em semântica, acessibilidade e boas práticas modernas.">
```

- **Ela:**

  1. **Não é fator direto de ranking;**

  2. **Afeta CTR (cliques);**
  
- **Boa descrição:**

  1. **140–160 caracteres;**

  2. **Clara;**

  3. **Persuasiva;**



### Headings bem estruturados

```html
H1 (1 por página)
 └─ H2
     └─ H3
```

- **SEO usa headings para entender hierarquia;**
  - **H1 deve conter palavra-chave principal;**
  - **H2 estruturam subtópicos;**
  - **...**



### Links internos

- **Exemplo:**

  ```html
  <a href="/guia-css">Guia completo de CSS</a>
  ```

  - **Boas práticas:**
    - **Texto descritivo;**
    - **Linkar páginas relacionadas;**
    - **Não usar “clique aqui”;**
  - **Links internos:**
    - **Distribuem autoridade;**
    - **Melhoram indexação;**
    - **Aumentam tempo no site;**



### URLS "amigáveis"

- **Ruim:**

  ```scss
  site.com/p=123?id=456
  ```

- **Bom:**

  ```scss
  site.com/curso-html5-completo
  ```

- **Regras:**

  1. **Minúsculas;**
  2. **Com hífen;**
  3. **Sem acentos;**
  4. **Descritivas;**



### SEO para imagens

- **Exemplo:**

  - **Nome do arquivo:**

    ```scss
    curso-html5.jpg
    ```

  - **`Alt` descritivo:**

    ```html
    <img src="curso-html5.jpg" alt="Aluno estudando HTML5 em notebook">
    ```

- **OBS.:**

  - **O Google também indexa imagens;**



### Performance

- **Boas práticas:**

  - **Imagens otimizadas;**

  - **Lazy loading:**

    ```html
    <img src="foto.jpg" loading="lazy" alt="Descrição">
    ```

  - **CSS minificado;**

  - **JavaScritp no final ou `defer`;**

- **OBS.:**

  - **O Google usa *Core Web Vitals* como fator de ranking;**
    - **O *Core Web Vitals* são métricas essenciais do Google que avaliam a experiência real do usuário (UX) em um site, focando em velocidade de carregamento, interatividade e estabilidade visual;**



### Mobile-first

```html
<meta name="viewport" content="width=device-width, initial-scale=1.0">
```

- **O Google indexa a versão mobile primeiro;**



### Conteúdo de qualidade

- **SEO moderno não é só técnica:**
  - **Responde intenção de busca;**
  - **Texto claro;**
  - **Parágrafos escaneáveis;**
    - **Blocos de texto estruturados de forma concisa e visualmente organizada, permitindo que os leitores da web identifiquem rapidamente as informações principais sem precisar ler palavra por palavra;**
  - **Listas ajudam;**
  - **Subtítulos organizam;**



### Dados estruturados (*Schema*)

- **Exemplo básico JSON-LD:**

  ```html
  <script type="application/ld+json">
  {
   "@context": "https://schema.org",
   "@type": "Article",
   "headline": "Guia de HTML5",
   "author": {
     "@type": "Person",
     "name": "Seu Nome"
   }
  }
  </script>
  ```

- **Ajuda o Google a gerar *rich snippets*;**



### Sitemap e robots

- **`sitemap.xml`;**

- **`robots.txt`;**

- **`Canonical` quando necessário:**

  ```html
  <link rel="canonical" href="https://site.com/pagina-original">
  ```

  - **Evita conteúdo duplicado;**



### Breadcrumb estruturado

- **Melhora SEO e UX:**

  ```html
  <nav aria-label="Breadcrumb">
    <ol>
      <li><a href="/">Home</a></li>
      <li><a href="/blog">Blog</a></li>
      <li>Post</li>
    </ol>
  </nav>
  ```



### E-E-A-T 

- **O Google valoriza:**
  - **Experiência;**
  - **Especialização;**
  - **Autoridade;**
  - **Confiabilidade;**
- **Incluir:**
  - **Autor identificado;**
  - **Página "sobre";**
  - **Contato real;**
  - **Política de privacidade;**



## Erros

- **Vários H1;**
- **Conteúdo duplicado;**
- **Keyword stuffing;**
  - **Prática de preencher um site com repetições excessivas e artificiais de palavras-chave para manipular o ranqueamento no Google;**
  - **Considerada técnica de "Black Hat" e spam, causa má experiência ao usuário e resulta em penalidades, como a perda de posições ou remoção do Google search;**
- **Texto escondido;**
- **Links quebrados;**
- **Site lento;**
- **Não usar HTTPS;**



## Considerações

- **CTR:**
  - ***Click-Through Rate* (em português, Taxa de Cliques)**;
  - **No contexto de SEO e Marketing Digital, é uma métrica que mede a frequência com que as pessoas que visualizam o seu site nos resultados de pesquisa realmente clicam nele;**
- **JSON-LD:**
  - ***JavaScript Object Notation for Linked Data* (Notação de Objetos JavaScript para Dados Ligados);**
  - **Formato leve de dados estruturados, recomendado pelo Google, que utiliza sintaxe JSON para conectar e contextualizar informações em sites;**
  - **Ele permite que motores de busca entendam melhor o conteúdo, facilitando a exibição de resultados ricos (Rich Snippets) sem afetar o front-end;**
- ***Rich Snippets*:**
  - **Resultados de pesquisa aprimorados no Google, exibidos com informações visuais extras, como estrelas de avaliação, preços, tempo de preparo de receitas ou imagens;**
- **Breadcrumb:**
  - **Sistema de navegação secundário que mostra a localização atual do usuário dentro da hierarquia de um site ou aplicativo;**
  - **Curiosidade:**
    - **O nome vem do conto de fadas "João e Maria", onde os personagens deixam rastros de migalhas de pão para encontrar o caminho de volta;**
